# Python To-Do List

A simple **terminal-based To-Do List** application built in Python.  
This program allows users to add, view, and remove tasks from a list. It's a beginner-friendly project to practice Python basics like lists, loops, functions, and input validation.

## Features

- Add a task to your to-do list
- View all tasks with numbering
- Remove a task by specifying its number
- Exit the program gracefully

## How to Run

1. Make sure Python is installed on your system.
2. Clone the repository or download the files.
3. Open a terminal in the project folder.
4. Run the program with:

```bash
python todo.py
